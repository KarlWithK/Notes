How I take notes using vim

Jun 21, 2020 Home

So I haven't found a more efficient way of writing notes for myself other than this, thus I wanted to share my setup.

I use

  • vim
  • vimwiki
  • coc-word (coc)
  • github pages

In addition I have the following in my vimrc: 

" plugins
Plug 'neoclide/coc.nvim', {'branch': 'release'} "not needed
Plug 'vimwiki/vimwiki' "To take notes better

"mappings to allow tab completion to work and I like it
nmap <Leader>ll <Plug>VimwikiFollowLink
nmap <Leader>ln <Plug>VimwikiNextLink
nmap <Leader>lp <Plug>VimwikiPrevLink
nnoremap <expr> j (v:count? 'j' : 'gj')
nnoremap <expr> k (v:count? 'k' : 'gk')
nnoremap <leader>z [s1z=
nnoremap <leader>5 :setlocal spell spelllang=en_us<cr>

"lets
let g:vimwiki_list = [{'path': '~/Work/vimwiki'}]
let g:vimwiki_table_mappings=0 "so I can tab complete with coc


let g:coc_global_extensions = ['coc-words']

I also have a snippet I like to use with ultrasnips, but I don't use it that much: 

snippet mdate "Date for me" b
${1:Name of article}
`!v strftime("%b %d, %Y")`
[[../index|Home]]
$2

endsnippet
