Malware: Malicious Software

Viruses, worms, torjans, rootkits

Malware can be classified into multiple categories, depending on propagation and concealment.

  • Propagation:
    • Virus: human-assisted propagation (open email attachment)
    • work: automatic propagation without human assistance
  • Concealment:
    • Rootkit: modifies OS to hide its existence
    • Trojan: provides desirable functionality but hide malicious operation.
  • Various types of payloads, ranging from annoyance to crime.

Propagation

Insider Attacks

An insider attack is a security breach that is caused or facilitated by someone who is part of the very organization that controls or builds the asset that should Br protected

Backdoors

Sometimes called a trapdoor, and it is a hidden feature or command in a program that allows a user to perform action they would normally not be allowed to do

  • When hidden feature is activated, the program does something unexpected, like giving more permissions to a user

Logic Bombs

A logic bomb is a program that performs a malicious action as a result of a certain logic condition.

  • Example is when a programmer codes the software for the payroll system so it has to pay him.
  • Or when a programmer codes in that the program crashes when it is reaches a certain day.

The Omega Bomb Code

A person destroyed his company by using a logic bomb on 7/30/96. Deleted critical information.

Defenses against insider Attacks

  • Avoid single points of failure
  • Use code walk-throughs
  • Use archiving and reporting tools
  • Limit authority and permissions
  • Physically secure critical systems
  • Monitor employee behavior

Computer Viruses

A computer virus is computer code that can replicate itself by modifying other files or programs to insert code that is capable of further replication

  • This property is what distinguishes computer viruses from other kinds of Malware.
  • Another feature is that replication requires.

Biological Analogy

Computer viruses share some properties with biological versus like replication.

Virus Phases

Many phases a virus goes in:

  • Dormant: just exists and avoids detection
  • Propagation phase: replicates itself
  • Triggering: some logical condition causes the birus to be activated.
  • Action phase: performs all actions it was designed to perform (payload):
    • could be innocent or not.

Infection types

  • Overwriting: destroys original code
  • Pre-pending: keeps original code, possibly compressed
  • Infection of libraries: allows virus to be memory resident like kernel32.dll
  • Macro viruses: Infects MS office documents (often installs in main document template).

Concealment

  • Encrypeted virus:
    • Virus that encrypts its payload to make it more hard to detect. However, an antivirus can use a derptor to deactivate it.
    • Polymorphic virus
    • Encrypted virus with random variations of the decrption engine
    • Detection using CPU emulator
  • Metamorphic virus:
    • A type of malware that is capable of changing its code and signature patterns with each iteration. Considered to more advance threats than typical malware or even polymorphic viruses.

Computer Worms

A computer worm is a malware program that spreads copies of itself without the need to inject itself.

Worm Propagation

Propagation by finding and infection vulnerable hosts.

  • Need a way to tell if a host is vulnerable or already infected
  • Propagation theory: Classic epidemic model
