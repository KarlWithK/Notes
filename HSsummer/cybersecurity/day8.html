Operating System Security

The boot sequence

The action of loading an operating system into memory from a powered-off state is known as booting or bootstraping.

  • When a computer is turned on, it first executes code stored in BIOS (basic input/output system)
  • BIOS loads into memory the second-stage boot loader, which handles loading the rest of the OS into memory and then passes control to the OS

BIOS Passwords

A malicious user could potentially seize exaction of a computer at server points.

  • To prevent this, many computers have a BIOS password that does not allow a second-stage boot leader without authentication.

Hibernation

Computers can go into a powered-off state known as hibernation

  • While in this state, the OS stores the contents of machine's memory into a hibernation file (such as hiberfil.sys) so it restore everything
  • Exposes a machine to potentially invasive forensic investigation.

Event logging

Logs keep track of all process are running, Internet aces, crashes: for the purpose of troubleshooting and determining the cause of a security breach.

Memory and Filesystem Security

The contents of a computer are encapsulated in its memory and Filesystem.

  • Thus, protection of a computer's contents has to start with the protection of its memory and Filesystem.

Password Security

The basic approach to guessing passwords from password file is to conduct a dictionary attack, where each word in a dictionary is hashed and the resulting value is compared with the hashed passwords in the password file.

  • A dictionary of 500k words is enough to discover most passwords

Password Salt

Use "salt", which is associated with a random number with each userID.

  • Rather than comparing the has of an entered password, the system compares the hash an entered password and the salt for the associated userid.

How Salt increases search space size

Increase search space for dictionary attack on a solted password is of size 2B*D. About 2 quadrillion. Where B is the number of bits of the random salt and D.
