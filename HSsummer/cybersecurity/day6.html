Physical Security

Any object that creates a barrier to unauthorized access like locks, doors, fences or guards

Destructive VS Nondestructive Entry

  • Destructive Entry:
    • Involves using force to gain access like a crowbar
    • Negative impact on IT resources is apparent
    • Remediation steps also obvious
  • Nondestructive entry:
    • Gain access without leaving signs
    • Defeat intrusion detection
    • Greater and long-term threat

    Pick Vs. Bypass

    Nondestructive to break a lock:

    • Pick: action on the lock mechanism simulating the operation of the key
    • Bypass: manipulation of the both without using the lock

    Yale Pin Tumbler Lock

    • made in 1860, and modern version of the Egyptian single-pin design
    • Utilizes two pins for locking
    • Double-detainer theory
    • Created Shear line
    • Works by alignment of the cuts of the key with shear line to route the plug
    • IN an inappropriate key is inserted, then it doesn't turn

    Tubular Lock

    • Usually on car alarms or vending machines
    • 608 pins
    • Easy to pick with special tool
    • The tool could become a new key.

    Lock Picking Tools

    Feelers, Scrubbers, and tension tools are all lock picking tools.

    Feeler Picking

    Apply light tension, and then lift pin by pin and then rotate.

    Scrubbing / Raking

    Apply tension, work over pins in a circular motion and then open it.

    • Good for beginners
    • Usually employ snake pick or half diamond.

    The Math of Lock Picking

    Given 30 different kinds of key blanks, 7 pin positions, and 8 different possible pin heights, there is 83.8 million combinations.

    • Not all these combination are possible due to the structure of a lock.
    • Meaning less secure.

    Rights Amplification in Master Key Systems

    1. Reverse engineer master key from change key
    2. Each lock has P pins, with D potential cut heights
    3. Create D-1 test keys for each pin position p
    4. Cut all pin positions except p as known change key
    5. Query the lock until you find each pin position
    6. Repeat for each pin
    7. Consumes P(D-1) blanks
    8. Search space is practically pruned by manufacturer specs.

    Bumping
