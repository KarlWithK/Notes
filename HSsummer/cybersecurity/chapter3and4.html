Basics of Windows Security https://youtu.be/Ed_2BKn3QR8 Overall this video was extremely concise, and well-made. It taught the basics of the security principal: who to authenticate and how to identify them - SID. A SID is an identifier that is formulated using a unique number. This is informant to allow multiple users to have the same name, but different data. The video explicitly states that it is better to disable a user than to delete a user as it allows transfer of permissions. Access Control Entry are part of an Access Control List that grants permissions to read / write to a file. These two items evaluate the user's SID to check if they have the correct permissions. The video finally showed that the way to secure the SID is to have an Access Token that prevents major attacks from using a fabricated SID. The user would need to relog in to allow another try at getting a token if the token was changed. The great thing about this video is that there were images and videos that showed exactly what was he was talking about. 9/ 10. A bit too long, but great examples.

Chapter 7 9 Intro to Malware https://youtu.be/OdjZ9p_TAWc Although the animation of this video are not as professional as the others, I thought it was well researched. It taught the definition of a malware - a "software that executes on your machine and has malicious effect on it". It describes several malware like: Trojans horses that pretend to do something else to be installed, back doors to gain access to a program without authentication, rootkits that deal on a kernel level, ransom ware that demands payment to unlock the computer, adware that use ads to attack the user, spyware that obtain all kinds of information, worms that duplicates themselves, botnets that take over a computer and crash it, and finally there is the classic virus. The narrator suggest to never download anything from a email and download an antivirus software that checks for signatures. The video was concise and very informational.
