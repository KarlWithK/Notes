Authentication

Determination of identity.

Barcodes

  • Made in 20th century for grocery checkout
  • First gen represents data as a series of variable-width, vertical lines of link. Encoding shame.
  • Later become 2D and have been used in airline
  • Provides convenience but not security as they are easy to duplicate

Magnetic Stripe Cards

Plastic card with a magnetic stripe containing information about the owner.

  • The first track of the card contains name of the cardholder and other data
  • The second track contains discretionary data.

Security

  • Vulnerable due to the medium being easy to read and reproduce
  • magnetic stripe reads can be purchased at low cost making it easy to read information
  • With magnetic stripe writer an attacker can easily clone a card.
  • PIN is needed in addition with the card

Smart Cards

Cards that have an integrated circuit with a microprocessor that allowing altering and reading of the data. Is better than magnetic cards.

  • Strong authentication that uses cryptography and used by large companies.
  • Electronic wallet. Debit card.

SIM Cards

SIM stands for Subscriber identity module card.

  • Issued by a network provider and contains information of the owner to allow access to a cellular network.

Security

  • Has a lot of information to authenticate to the approbate network
  • Network provider has a record of the card
  • Features Integrated Circuit Card ID (ICCID) which is a 18-digit number used for hardware identification.
  • Has a international mobile subscriber identity (IMSI) -> more info about owner
  • Contains a 128-bit secret key that is use for authentication.
  • Some have a pin to access all the data.

GSM Challenge-Response Protocol

  • Cellphone connects to a local base station and transmits its IMSI to connect to a cellular network.
  • The phone will recieve a 128-bit random number to the cellphone, which is ended ed by the SIM Card's secret key using an algorithm called A3.
  • Results in a ciphertext that are computed in both the phone and the base station and if they match the cell phone is allowed to make calls.

RFIDs

Radio frequency identification (RFID) relies on small transponders to transmit identification information via radio waves.

  • RFID chips have an integrated circuit and a coiled antenna to transmit and receive a radio signal.
  • RFID tags must be used with a separate reader or writer. Some which may need a battery
  • Range of RFID varies but it most cases a tag doesn't need to be closed to the reader.
  • Many applications like consumer-product tracking, Car key fobs, or electronic toll transponders.
  • Passports have an RFID chip that contains a digital facial photograph:
    • RFID messages are also encrypted with a secret key
    • Usually the key is the passport number, the holder's date of birth, and the expiration date. -> Usually on the passport itself since the key is intended to be accessible physically only
    • However an attacker can take this information using social engineering.

Biometrics

Any measure used to identity a person based on biological traits.

  • Contains a sensor or scanner to read in information and compare it to a database of accepted users.

Requirements

  • Universality: every person should have this characteristic
  • Distinctiveness, each person should have differences on said traits
  • Permanence: Trait should not change over time.
  • Collectability: Should have some quantitative aspect and should be determine quickly.
  • Biometric goes through the reader and is converted into a feature vector, which is compared with a reference vector.

Candidates for biometric IDs

  • Fingerprints
  • Retina
  • DNA
  • ‘Blue ink” sign.
