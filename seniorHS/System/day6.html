  • Array Basics:
    • An array is an allocated block of memory meant to hold multiple piece of data of the same type.
    • C arrays not use have a length attribute/function
    • WE will use [] to access array elements.
    • The size of an array must be set at declaration and cannot be changed.
    • The size of an array cannot be dynamic.
      • Importantly, this means you cannot set the size of an array to be result of a function or other expression that will not be evaluated until the program is run.
    • There is no boundary checking (much more on this later).
    • Array declaration/access syntax

float ray[5]; ray[2] = 0.22 // enough space for 5 floats.

-The above code request a block of memory large enough for 5 floats (20 bytes) which then can be accessed using 0-based [] notation.
