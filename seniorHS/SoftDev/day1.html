    Virtual Environment recap:

    • Allows you to maintain a specific python environment without messing with the system version
    • Allows you to install packages without root access to a system
    • Creates environment w/ its own install dirs
    • Does not shape libs w/ other virtual envs
    • Practically "sandbox" for your python code.
