Stack

  • A stack is a collection of objects following the last-in-first-out (LIFO) principle. We put to the 'top' of the stack by a "push" and 'pop's removes top of the stack.

;

  • Simple applications:
    • reversal of Strings, 'undo'feature, backtracking (exaole: move solving), parsing.
  • Implement these methods.
    • public void push(String x) // Push a string onto the stack, or another object of your choice
    • public Strings pop() //remove the 'top' String from the stack;
    • public String peek() // look at the 'top' String from the stack, but do not remove
    • public boolean isEmpty() // checks to see if the stack is empty.

Queues

  • Queues follow a first-in-first-out (FIFO) principle. SOmething that is enqueue'd will be put into the list at the end. Something that is dequeue'd will be removed from the front of the list.
