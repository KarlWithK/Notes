Inheritance is creating a new class from an existing one. For example, "Animals", can describe alive, and you can assign an "age" to such an object. This variable can be inherited to all objects derived from "animals"

All public variables and public methods will be inherited by derived class. Private variables and private methods will not be inherited. Protected variables or methods will be inherited, but cannot be directly assessed.

Superclass, or base class. The derive class is called child class, subclass.

The major reason why we use inheritance, is to reuse code.

We use the keyword, extends in the child to create an -is-a relationship between parent and child. 

public class childClass extends parentClass

The reserved word, super, is used to reference the parent to reference the parent class from child class.
