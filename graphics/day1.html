Other color spaces

RGBA

3 bytes for red green and blue. 3 bytes is a weird number not a power of two.

Hardware uses 4 bytes. -> RGBA (Red, Green, Blue + Alpha(Transparency))

Makes the image not look like a rectangle. Jpeg doesn't support this file format

HSB

No red, green and blue format

Stands for hue, saturation and brightness

Depicts a cone

Brightness is Z level, hue is x and saturation is Y

Image File formats

Raster vs. Vector

Vector

  • Vector formats represent images as a series of drawing instructions
  • Infinitely scalable
  • We will use it.
  • SVG (Scalable Vector Graphics) - well known vector format
  • Not a lot of data to represent an image = Raster =
  • Most monitors are Raster.
  • Raster formats represent images as a rid of pixels.

Uncompressed vs. Compressed (Raster)

Uncompressed

  • Uncompressed formats contain data for each pixel.
  • DNG raw images
  • BMP, TIFF, RAW
  • File size huge! = Compressed =
  • Compressed formats use a compression a algorithm to minimize file size
  • Lossless vs Lossy
    • Lossless compression algorithms contain enough information to exactly recreate the ordinal image
      • run length encoding - just find a pattern
      • PNG (portable Network Graphics), GIF (Graphics Interchange Format)
    • Lossy compression algorithms not retain all the details of the ordinal image.
      • JPEG (Joint Photographic Experts Group)
