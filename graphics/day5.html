Matrix Multiplication

  • M0* M1 != M1 * M0
  • Number of Columns in M0 needs to equal number of rows in M1
  • for example [1 2 3] * [a , b , c] = [1a + 2b + 3c] -> A single value
  • 1x 3 = 3x1
  • Edge Matrix = 3 x N -> should be on the left
  • Our resulting matrix size should be 3XN
  • (Will really be 4N)

Multiplicative Identity

  • I * M0 should be equal M0
  • 1 0 0 * x0 x1 = x0 x1
  • 0 1 0 * y0 y1 = y0 y1
  • 0 0 1 * z0 z1 = z0 z1
