Parametric Equations

  • Any curve ( system of equations ) where the x and y values depend on a third variable (usually t or time) to make the relationship not direct between x and y.
  • Define curves as system of equations of equations in terms of a single independent variable (t).
  • Allows us to draw different shape with the same framework
  • Non Parametric:
    • y = f(x)
  • Parametric:
    • y = f(t)
    • x = q(x)
    • two different functions which are linked by two different functions
  • T can be thought as a percentage. Midway through the curve can be thought as t = 0.5
  • Parametric line:
    • t is another variable
    • (x0 , y0) -> (x1 , y1)
    • X = x0 + (x1 - x0)t
    • y = y0 + (y1 - y0)t
    • t0 = 0
    • x = x0
    • y = y0
    • t1 = 1
    • x = x0 + delta x
    • y = y0 + delta y
  • Circle:
    • x = rcos0 + Cx (center x) => rcos(2pi * t) + Cx
    • y = rsin0 + Cy (center y) => rsin(2 pi * t) + Cy
    • 0 : 0 - 2pi
