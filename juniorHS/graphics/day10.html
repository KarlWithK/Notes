Splines

  • Curves (cubic for the purposes of this course) that can be connected to appear smooth / continuous
  • Hermite Curves:
    • Defined by two endpoints (p0,p1)
    • Rates of changes at each endpoint (R0,R1)
    • f(t) = at3 + bt2 + ct + d
    • f1(t) = 3at^2 + 3bt + c
    • f(0) = d = p0
    • f1(0) = c = R0
    • f(1) = a + b + c + d = p1
    • f1(1) = 3a + 2b + c = R1
    • a = 2p0 - 2p1 + R0 + R1
    • b = -3p0 + 3p1 - 2R0 - R1
    • c = R0
    • d = p0
