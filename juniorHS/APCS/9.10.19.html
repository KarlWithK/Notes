Man memory and secondary memory:

Main memory is made up of many consecutive locations, called cells.

A cell can be referenced, accessed, and modified inf ideally. A memory wit a low address is just as accessible as a high address. Memory can be processed in any order, or random accessed.

Storage capacity is the number of bytes it can hold. Units of storage in powers of two.

Bit 1 b 1/8 of a byte Byte 1B 2^0 = 1 Megabyte GB 2^30 - 1, 073, 741, 824 Terabyte TB 2^40 = 1, 099, 511, 627, 776

Main memory is also called RAM (random access memory). Larger RAM allows a computer to retrieve information from secondary memory less frequently.

HT <-> Slow Transfer {MAIN MEMORY} Faster Transfer <-> CPU

Main memory is considered volatile, because the information will be lost when there is lost in power. Secondary memory are usually non-volatile, where the information is stilled retained where there's no power.

Most common secondary memory are hard drives (40 to ~160+ GB), floppy disks (1.44MB), CD-R,DVD-R, memory sticks

Secondary memory (Magnetic Disk): A hard drive consists of magnetic disks, where a bit is represented by a magnetized particle on the disk. A read/ write head passes over the spinning disk.

A disk is divided into tracks. Tracks are divided into arcs, called sectors. The Read/Write head "floats" just above the surface. The disk spins at speeds at 3000-8000Ron. There is also a seek time for the head to find the appropriate track. High capacity disk systems consist of many platters on a common spindle.

RAM VS ROM:

RAM has the ability to read, write, and rewrite. ROM (Read only memory) can be read once the data is written. -Microchips can be ROM, embedded with instructions. -CD-R (700MB) are similar to magnetic disks, except they have microscopic physical pits, which when read by a laser when reflected back. Physical pits, allows mass production of Cads such as music. -CD-AW(700MB) use special material that is heat sensitive the same effect. Operates like a CD-R, but can be changed. -Dads (4.7-8.5GB) have more bits per square inch than a CD, allowing more data to be stored. -Magnetic <tapes></tapes>sun as cassettes or high-capacity backup systems, use one long strip of magnetic tape, with a read/write head and two reels.

Main memory <-> Bus <-> CPU (Control unit, arithmetic/logic unit, registers

Registers provide a temporary steerage of information. They can be classified as either general-purpose registers or special-purpose registers.

Also a kind of memory called cache (might be located next to the CPU). Places to store stuff. La > la > registers. Communication needs to be fast. Less time to look at things.

Cache depends on different kinds of address locality to be efficient. Spatial locality: consecutive cell addresses are more likely to be addressed. Temporal locality: recent addresses will be accessed.
