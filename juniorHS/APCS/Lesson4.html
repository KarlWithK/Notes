Aim: Sorting Arrays (6.3)

Sorting is simply arranging a list of items in order.

Selection sort Putting items into their final, sorted positions, is the selection sort algorithm.

  1. Scan the list and find the smallest value.
  2. Swap the value with the first position.
  3. Repeat on the remaining set of items.

Insertion sort ( start with two items )

  1. Compare a new item with all items in the all items in the new list, from right to left, shifting all greater items to its right.
  2. Create an empty slot, and insert the new item into the open slot.
  3. Repeat on the remaining set of items.
