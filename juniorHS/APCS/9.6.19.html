A computer system consist of hardware and software. Ex: Hardware: Monitors, Mouse and Keyboards

Software: Windows and data files

Key Hardware components:

  • Central processing unit (CPU)
  • Input/Output (i/O) Devices
  • Main Memory: storage device the holds the program while the CPU executes it (RAM)
  • Secondary memory devices Physical devices that can hold data for long periods

CPU <-> Monitors ->Main memory <-> Hard disk

Software Categories

Operating system (OS)

  • Provides a user interface.
  • Manages computer resources such as the CPU, memory, programs, and hardware device communications

Application: A term for any software other an OS.

Graphical User Interface (GUI): Is any interface that is simplified using graphics

Ex: windows - distinct work areas, icons, buttons.

Non-GUI: Dos operating system, Terminal

Digital vs Analog

Analog: continuous information Digital: Information in pieces

Sound is stored digitally by taking samples at constant intervals: Called sampling devise

All information is devitalized. Ex: Text - Unicode

The way a computer stores these 'numbers' is in binary. A digital signal will have two extremes,"high" (1) and "low" (0).

binary notation has only two possible values, 1 and 0, which are represented by bits.

A bit basically represents "on" or "off".

A byte, is 8 bits together, allowing 2^8 different permutations.

A megabyte, is about 1 Million bytes.

A gigabyte, is about 1 thousand megabytes.
