Instance variables are variables declared in class but not methods Local variables area variables declared in methods but not in class. Static variables are variables that affects all objects.

Constants (final modifier) are usually declared using the static modifier, since it can't be changed, one copy is enough.

Static methods

Can be called without creating an instance of object. Like the math class there is no need to create an object.

The main method is also static, so that we can run the program without needing a main object.

static methods cannot reference instance variables a compiler error will occur..
