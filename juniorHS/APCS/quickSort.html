Aim: Recursion with Sorts(11.x)

Quick Sort [0 (n log n)]

There are several ways to do this

  1. Always pick first element of array.
  2. ALways pick last element as pivot
  3. pick a random element as pivot
  4. pick median as pivot 
	public static int[] quickSort(arr[], low, high) {
		if (low < high) {
		// pi is paritining index
			pi = partition(arr, low, high);
			quickSort(arr, low, pi - 1); //before pi
			quickSort(arr, pi + 1, high); // after pi
		}

	}

	public static void partition(int[] arr, int low, int high) {
		// pivot
		pivot = arr[high];

		index = low;

		for (int j = low; j < high; j++) {
			// if current element is smaller than the pivot
			if (arr[j] < pivot) {
				//swap arr[i] and arr[j]
				i++ // increment index of smaller element
			}
		}

		swap arr[i] and arr[high]
		quickSort everything before the index and quicksort everything after the index
	}

Partition method will select a pivot element, and then place all elements less than the pivot to the left, and all elements greater to the to the right
