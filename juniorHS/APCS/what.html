All classes are derived from the object class, even if it doesn't use extends.

Some methods:

  • equals(Object obj)
  • clone()
  • .toString()
  • .hashCode()

Abstract class: An abstract class uses the abstract keyword modifier. It is similar to an interface, in that you cannot create an object from it, as some of its methods may not be completed yet, it can contain data declarations and constants APCS
