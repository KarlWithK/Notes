Aim: Recursion (11.1)

Recursion requires two ideas:

  1. Base Case,
  2. Recursion definition

For example: A list of numbers:

  1. A number,
  2. Or comma followed by another list.

Factorial (n!. Where N (integer) >= 0):

  1. 1 If N == 1
  2. N * (N - 1)

Recursive code to sum 1 + 2 + ... N: 

	public	int sum (int num) {
		int result;
		if (num == 1)
			result = 1;
		else
			result = num + sum (num - 1);
		return result;
	}

Recursion vs iteration:

Recursion, should we used to solve problems in a more elegant fashion. Iteration should be used when the solution isn't complicated.

Infinite recursion can lead to memory issues, as infinite iteration use CPU cycles.

Recursion code is shorter. While iteration is longer. You can always make a recursive algorithm into a iterative algorithm and vice versa. APCS
